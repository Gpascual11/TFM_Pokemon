# p04_backup: Project Archive

This directory contains historical data, early experiments, and analysis notebooks that are no longer part of the "hot" development path but remain valuable for reference.

## Content

- **`heuristics/`**: All the analysis notebooks used during the initial design of the rule-based agents.
  - `analysis.ipynb`: Win-rate plots for different heuristic versions.
  - `type_analysis.py`: Early experiments with type effectiveness matrices.
  - `analysis_v1_vs_v2.ipynb`: Comparative study of the transition from independent to joint-action scoring in doubles.

---

## Usage

These files are primarily for documentation and research history. Most code here may require legacy imports or specific environment snapshots to run perfectly, as the main project structure has since evolved.
