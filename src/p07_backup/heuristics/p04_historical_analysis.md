# heuristics: Historical Analysis Archive

This folder stores the logs and Jupyter notebooks used to design the v1-v3 heuristic strategies.

## Contents
- **`analysis.ipynb`**: The primary research notebook for singles.
- **`analysis_doubles.ipynb`**: Research on the 2v2 joint-action space.
- **`type_analysis.py`**: Performance analysis of different Pokémon types (STAB vs non-STAB).
- **`analysis_utils.py`**: Matplotlib and Seaborn helpers for plotting win-rate heatmaps.
